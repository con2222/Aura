# Aura - Top-Down RPG (Unreal Engine 5)

**Статус проекта: В активной разработке (Work in Progress)**
*Игра находится на стадии создания, механики добавляются, а код регулярно обновляется.*

## О проекте

**Aura** — это ролевая игра с видом сверху (Top-Down Action RPG), разрабатываемая на движке **Unreal Engine 5**.

Главная цель этого проекта — глубокое изучение и практическое применение мощного плагина **Gameplay Ability System (GAS)** от Epic Games.

## Основные механики (реализованные или в планах)

* **Архитектура GAS:** Полная настройка базовых классов на C++ (Ability System Component, Attribute Set).
* **Система атрибутов:** Здоровье, мана, броня и другие характеристики, управляемые через Gameplay Effects.
* **Способности (Abilities):** Кастование заклинаний, снаряды, расчет урона и затрат ресурсов.
* **Интерфейс (UI):** Динамические полоски здоровья и маны, всплывающий урон (Floating Damage Text), виджеты.
* **Враги и AI:** Искусственный интеллект противников, логика поведения и атак.
* **Прогрессия:** Система получения опыта и повышения уровня (Level Up).

## Технический стек

* **Движок:** Unreal Engine 5
* **Основа:** C++ и Blueprints
* **Ключевой плагин:** Gameplay Ability System
